# Classic Snake

Classic Snake game with TypeScript and ADT design.

Monorepo with 3 packages: `@snake/shared` (game logic), `@snake/client` (browser UI), `@snake/server` (API).

## Quick Start

```bash
pnpm install
pnpm build

# Run client
pnpm dev:client

# Run server
pnpm dev:server

# Run tests
pnpm test
```
